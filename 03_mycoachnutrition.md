# MyCoachNutrition

MyCoachNutrition is an LLM-powered SaaS product built and deployed by Jérémie.

It is a live AI product used by real users.

The product generates personalized weekly nutrition plans based on user inputs, goals and progression data.

## What Jérémie built

- Account management
- Subscription logic
- Prompt logic
- Direct LLM API integration
- Automated plan generation
- Scheduled weekly plan updates
- Error handling
- Retries
- Fallbacks
- Production safeguards
- Monitoring of generated outputs and user feedback

## User problem

Many users want personalized nutrition guidance but struggle to translate goals, constraints and preferences into a realistic weekly meal plan.

## Product decision

Build an AI-powered system that generates personalized weekly plans and adapts them over time based on user inputs and progression data.

## AI workflow

The system uses structured prompts, direct API calls, automated generation logic, scheduled updates and reliability safeguards.

## Production learning

Jérémie monitored generated outputs and user feedback to identify edge cases, improve prompt logic and reduce recurring failure modes.
