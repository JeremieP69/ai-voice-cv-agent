# AI Voice CV Agent

A voice and chat AI agent designed to help recruiters explore my background, projects and fit for AI Product / Technical Solutions roles.

The agent is grounded in a structured knowledge base covering:
- my professional background
- Truckfly by Michelin
- MyCoachNutrition
- AI workflow experiments
- BMAD agentic product development workflow exploration
- product, technical and customer-facing achievements

The goal is not only to present my CV, but to demonstrate a practical use case of voice AI, agents, knowledge-base design and conversational UX.
